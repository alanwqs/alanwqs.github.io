# 一、网络基础TCP/IP

# 1、TCP/IP协议族

TIP

  • 双方进行通信，必须基于相同的方法，使用同一种语言，决定哪方发起/结束，不同系统之间的通信等
  • 互联网相关联的协议集合起来称之为TCP/IP协议，从电缆到显示网络资源中用到的协议

# 2、TCP/IP分层管理

TIP

  • 应用层
  • 传输层
  • 网络层
  • 数据链路层

应用层

决定向用户提供通信时提供的服务，比如常见的FTP文件传输协议，DNS域名查询系统，HTTP协议也是位于该层

传输层

传输层对上是应用层，提供两台计算机之间的数据传输，比如常见的协议有TCP协议，UDP协议

网络层

处理网络上流动的数据包，网络传输的最小单位，规定通过怎样的路径到达对方的计算机，把数据给对方，就是从网络世界里指定一条路，比如IP协议。

数据链路层

链接网络的硬件部分，包括网卡、光纤等物理可见的部分

# 二、IP/TCP和DNS协议

# 1、负责传输的IP协议

IP协议作用就是把各种数据包传输给对方，其中两个重要条件就是IP地址和MAC地址。其中IP指明了电脑节点被分配到的地址，是动态的。MAC地址是网卡所属的固定地址，基本不会更改。

通信过程中会数据包会进行多次的中转，会利用下一站中转设备的MAC地址来搜索下一个中转目标，此时就会用到ARP协议

ARP地址解析协议

能够根据通信方的IP地址反查出来对应的MAC地址

路由机制

到达通信方之前，网络中的计算机和路由器只能粗略的知道传输路线，没有谁能够全面掌握网络中的传输情况。

# 2、确保可靠的TCP协议

  • 将大块数据进行拆分成报文段，通过三次握手策略进行双方链接
  • 数据包发送之后不会置之不理，而会向对方进行确认。（SYN，ACK）
